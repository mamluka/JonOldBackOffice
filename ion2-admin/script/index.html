Benefits

  • Easy date input - With Tigra Calendar date input is simple and intuitive. Visitors can select any date from unlimited range with few clicks.
  • Cost Saving Component - Don't waste your time coding date conversion and validation all over again for each new project. Get the dates on server side ready for further processing.
  • Professional Look - Simple great looking improvement to the user interface of your web application.

Tigra Calendar PRO additionally offers:

  • Support for ANY custom date or time format
  • All aspects of appearance (colors, font, spacing etc.) are easy configurable; existing CSS properties of the web site can be linked to the calendar elements
  • Date limits (minimum and maximum allowed dates) are supported as either absolute date or relatively in days from today
  • Quick years and months scrolling with drop-down boxes, next/previous buttons are also supported
  • Reset date to today button
  • List of holidays or other dates requiring special highlighting (for example those not available for booking) supported
  • Forbidden dates list (by date, by day of week etc)
  • Allowed dates list (blacks out all the dates except those listed)
  • Configurable behavior on out of range date selection
  • Free setup support, free product updates, discounts for other products
  • Inline, pop-up and modal mode versions of the calendar
